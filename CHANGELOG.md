## [1.1.0] - 2026-06-27

### Added

- Added dedicated About, Services, Stack and Contact pages.
- Added expanded homepage sections for services, project showcase, FAQ, work process and final CTA.
- Added a project hub with explorer, delivery dashboard, comparison matrix, production gate and maintenance guide.
- Added dynamic project case study pages.
- Added project technical overview, implementation details, testing review, release status, asset manifest and roadmap sections.
- Added live demo, source code and preview status handling for projects.
- Added centralized project link resolver.
- Added centralized navigation configuration.
- Added visual project previews for portfolio presentation.
- Added JSON-LD structured data for key pages.
- Added sitemap, robots, manifest and Open Graph routes.
- Added custom root not-found page.
- Added localized not-found page.
- Added localized error boundary.
- Added production readiness checklist on the Stack page.
- Added documentation for adding new portfolio projects.
- Added reusable project entry template documentation.
- Added production checklist documentation.
- Added deployment options documentation.
- Added release metadata configuration.
- Added visible release version in the footer.
- Added final release notes and handoff documentation.

### Changed

- Redesigned the homepage with stronger Z-TECH positioning.
- Reordered homepage sections for a clearer business and portfolio flow.
- Expanded the Services page into a more complete offer page.
- Redesigned project cards, featured project section and project case study pages.
- Reordered project pages so live demo, source code and preview appear before deeper technical documentation.
- Reordered the Projects page so project entries appear before internal production and maintenance sections.
- Improved Header and Footer maintainability through shared navigation configuration.
- Improved production hardening with fallback pages and release checks.
- Finalized the portfolio as a production-ready system for adding future projects.
